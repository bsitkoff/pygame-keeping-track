## Try It Out
 
Run `track.py` by typing `python3 track.py` in the terminal and pressing Enter.

This game has three **states**: a start screen, the game itself, and a game over screen. Play through the whole cycle.

**Notice:**
- How do you start the game? How do you restart it?
- Where does the score display?
- What triggers the game over?

Now look at the code. Find the `state` variable and trace how it gets checked in both `update()` and `draw()`.
